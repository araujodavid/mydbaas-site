MyDBaaSMonitor

A web environment for Database-as-a-Service monitoring. This application was developed using the MyDBaaS framework.

  • Open source
  • Extensible
  • Easy to use
  • Clean layout. Some screenshots can be seen here.

The application is in a repository on GitHub. You can download the zip file or do the fork. Import the project in Eclipse and add the Tomcat server. To put into production just export the WAR file and then deploy.
